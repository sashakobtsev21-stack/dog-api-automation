# Dog API Automation Framework

Проект по автоматизации тестирования [The Dog API](https://thedogapi.com/).

Технологический стек
Python(requests)
Pytest(фикстуры, conftest)
Allure(отчетность) в процессе изучения

#Структура проекта
`api/` — транспортный слой и бизнес-логика запросов.
`tests/` — тестовые сценарии и конфигурация Pytest.

Запуск тестов
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите тесты: `pytest tests/`
